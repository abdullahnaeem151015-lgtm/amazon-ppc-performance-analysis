# amazon-ppc-performance-analysis
Analyzed Amazon PPC campaign data to identify inefficiencies and optimize performance. Using data analysis, trend evaluation, and performance metrics (ACOS, ROAS, spend, conversions), I reduced ACOS from 34.6% to 21.6% while increasing revenue and orders, demonstrating data-driven decision making and marketing analytics skills.
Used Amazon search term reports and search query performance to gather insightful data and make data driven decisions based on that.
Tools and platforms used for this project was Excel, google sheets, Amazon advertising console and Amazon seller central.
